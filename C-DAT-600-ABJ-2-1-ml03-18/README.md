## 📖 Introduction
Troisième partie (**ml03**) de la série **C-DAT-600 (Advanced Machine Learning)**. Le focus passe ici sur les méthodes d'ensemble (Ensemble Learning).

## 🎯 Objectifs
- Découverte et utilisation des algorithmes Random Forests et Gradient Boosting (comme XGBoost).
- Optimisation par combinaison : Comment l'agrégation de multiples modèles faibles permet-elle d'obtenir de fortes prédictions et d'éviter l'overfitting.
- Visualisation de l'importance des variables (Feature Importance).

## 📂 Structure
- `ML.ipynb`, `ML_04.ipynb` (brouillon de préparation) : Fichiers d'expérimentation avec Random Forests sur des jeux variables de dimensions croissantes.

## ⚙️ Exécution
```bash
jupyter notebook ML.ipynb
```
