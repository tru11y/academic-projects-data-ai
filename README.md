Ce dépôt regroupe l'ensemble de mes projets académiques réalisés dans le cadre de ma formation **We.Code (Epitech)** et de ma **Licence en Développement d’Applications**.  

Il illustre concrètement ma progression et mes compétences en **Programmation Python, SQL, Power BI, Machine Learning, NLP, Big Data et Data Visualization**. Chaque projet a été pensé pour répondre à des problématiques techniques ou analytiques spécifiques, allant de la manipulation de données brutes à l'implémentation de modèles prédictifs avancés.

---

## 📂 Architecture et Liste des Projets

### 1. 🐍 Python Basics (`C-DAT-100-*`)
Série de projets visant l'acquisition d'une base technique solide.
- **[py01 à py05]** : Exercices fondamentaux en Python.
- **Concepts clés** : Manipulation de données, programmation orientée objet (OOP), algorithmes basiques, structures de contrôle, gestion des exceptions.

### 2. 📊 Data Visualization (`C-DAT-200-*`)
Projets axés sur l'art de raconter des histoires avec les données (Data Storytelling).
- **[dataviz]** : Création de visualisations interactives exploratoires.
- **Outils** : Matplotlib, Seaborn, Plotly.
- **Objectif** : Transformer des données complexes en insights visuels clairs et actionnables.

### 3. ⚙️ End-to-End Machine Learning (`C-DAT-300-*`)
Pipeline complet de développement d'un modèle d'apprentissage.
- **[endtoendml]** : Processus comprenant l'extraction, le nettoyage (preprocessing), l'entraînement, l'évaluation et le déploiement du modèle.
- **Objectif** : Maîtriser le cycle de vie complet (Lifecycle) d’un projet Machine Learning.

### 4. 🧠 Natural Language Processing (NLP) (`C-DAT-500-*`)
Exploration du traitement du langage naturel.
- **[nlp]** : Classification de sentiments, analyse sémantique, tokenization, et modélisation TF-IDF.
- **Objectif** : Appliquer les techniques de machine learning aux données textuelles non structurées.

### 5. 🤖 Machine Learning avancé (`C-DAT-600-*`)
Plongée en profondeur dans les algorithmes prédictifs.
- **[ml01 à ml05]** : Implémentation de modèles supervisés et non-supervisés (Régression, Random Forest, SVM, Réseaux de neurones simples).
- **Techniques** : Cross-validation, GridSearch pour l'optimisation des hyperparamètres, évaluation métrique rigoureuse.

### 6. 🌐 Big Data (`C-DAT-700-*`)
Sensibilisation aux enjeux du traitement à large échelle.
- **[bigdata]** : Manipulation de grands volumes de données, exploration heuristique et calcul de KPIs complexes.
- **Objectif** : Démonstration de scalabilité et de traitement distribué.

### 7. 🏆 Projet de Synthèse (ECP) (`C-DAT-900-*`)
Le projet final intégrant les compétences acquises au cours du cursus.
- **[ecp]** : Application web complète (Streamlit/Flask) incluant data pipelines automatisés, dataviz et analyse business pointue.
- **Objectif** : Démontrer une capacité de bout-en-bout à traduire une problématique technique en valeur ajoutée métier.
