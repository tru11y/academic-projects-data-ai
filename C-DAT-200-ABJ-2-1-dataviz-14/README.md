## 📖 Introduction
Ce projet fait partie du module **C-DAT-200 (Data Visualization)**. Il met en pratique la **Visualisation de Données (DataViz)** essentielle pour explorer et communiquer des résultats statistiques issus d'un dataset de manière claire et percutante.

## 🎯 Objectifs
- Effectuer une **Exploratory Data Analysis (EDA)**.
- Maîtriser les différents types de graphiques (histogrammes, nuages de points, boxplots, graphiques interactifs).
- Analyser la distribution, la corrélation, et les valeurs aberrantes (outliers) au d'un jeu de données complexe.

## 🛠️ Stack Technique
- **Python** (Scripts et Notebooks)
- **Pandas** : Préparation et nettoyage des données (Data wrangling).
- **Matplotlib / Seaborn** : Graphiques statiques exploratoires de haute qualité.
- **Plotly** : (Optionnellement) Graphiques interactifs orientés dashboarding.

## 📂 Structure du projet
- `dataviz.ipynb` : Le bloc-notes Jupyter détaillant chaque étape visuelle de l'analyse (chargement, nettoyage, tracés graphiques avec observations markdowns).
- `dataviz.py` : Script Python autonome pour générer et enregistrer les représentations visuelles clés.

## ⚙️ Exécution
Vous pouvez exécuter l'analyse depuis le notebook via Jupyter :
```bash
jupyter notebook dataviz.ipynb
```

Ou exécuter le pipeline de génération visuelle en script :
```bash
python dataviz.py
```
