## 📖 Introduction
Ce projet fait partie du module **C-DAT-100 (Python Basics)**. L'objectif de la série `py01` est d'acquérir les fondamentaux du langage Python au travers d'exercices progressifs (variables, conditions, boucles, fonctions).

## 🚀 Fonctionnalités
- Manipulation d'arguments de ligne de commande `sys.argv`.
- Création de fonctions simples (calculs, affichages).
- Découverte de l'écosystème Python (imports, scripts autonomes).
- Premières expérimentations avec les expressions régulières et la manipulation de chaînes de caractères `string`.

## 📂 Structure
- `ex_01` à `ex_12` : Contient des scripts axés sur différents concepts de base. Par exemple le comptage de mots, l'incrémentation, ou l'affichage de données.

## ⚙️ Exécution
```bash
python ex_01/script.py
```
