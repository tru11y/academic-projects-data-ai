## 📖 Introduction
Le module `py02` introduit des concepts plus avancés de programmation, tels que la **Programmation Orientée Objet (POO)** en Python.

## 🚀 Fonctionnalités
- Création de classes et d'instances (`Transaction`, `Budget`, `History`).
- Utilisation de modules et packages (`__init__.py`).
- Héritage, polymorphisme et encapsulation.

## 📂 Structure
- `ex_01/Transaction.py` : Modélisation d'une transaction financière.
- `ex_02/Budget.py` : Gestion basique d'un budget.
- `ex_03/History.py` : Historisation des événements ou transactions.
- `ex_04/Finance/` : Package Python regroupant des classes financières.

## ⚙️ Exécution
```bash
python ex_04/test.py
```
