## 📖 Introduction
Quatrième volet (**ml04**) de la série **C-DAT-600 (Advanced Machine Learning)**. Ce module aborde le sujet critique de l'**Optimisation et du Fine-Tuning de Modèles**.

## 🎯 Objectifs
- Utiliser les grilles de recherche par hyperparamètres (`GridSearchCV` et `RandomizedSearchCV`).
- Appréhender avec précision le processus de validation croisée (`Cross-Validation`, K-Fold).
- Maximiser la précision prédictive tout en limitant les temps de calculs sur des instances volumineuses.

## 📂 Structure
